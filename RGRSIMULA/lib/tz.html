ТЕХНІЧНЕ ЗАВДАННЯ


на виконання курсового проекту з дисципліни «Моделювання»


     Тема: Моделювання роботи для автомобілістів, Варіант «AvtoBank»

     Виконавці: Жигульський Р.М, Прядко В.В, Мироненко О.С гр.КС-111

     Опис системи

     У банку для автомобілістів є кілька віконець, кожне з яких обслуговується одним касиром і має окрему під'їзну смугу. Смуги розташовані поруч. Автомобілі прибувають через випадкові інтервали часу. Клієнт вибирає смугу з найкоротшою чергою. Довжина смуг обмежена. Якщо всі місця на смугах зайняті, то черговий клієнт вважається загубленим. Після обслуговування, клієнт виїжджає на єдину смугу для виїзду з банку, але проблема полягає в тому, що автомобіль може потрапити на дорогу тільки з появою певного інтервалу між їдучими по ній автомобілями. Якщо на смузі для виїзду немає вільних місць, автомобіль не може від'їхати від віконця касира й обслуговування припиняється.

     Завдання на проектування

     Створити Java застосування для імітаційного моделювання робіт з обслуговування клієнтів, яке позволить:

  • перегляд технічного завдання;
  • налаштування параметрів моделі, а саме: тривалість моделювання, кількість смуг, кількість місць на смузі, кількість місць на виїзді, закони розподілення для випадкових величин інтервалу прибуття клієнтів, часу обслуговування, часу виїзду, часу зайнятості дороги;
  • проведення тестових запусків моделі при різних налаштуваннях з динамічною індикацією черги на обслуговування, черги на виїзд, втрачених клієнтів, та можливості виводу протоколу роботи моделі під час тестових запусків;
  • проведення експериментів для отримання статичних характеристик для часу черги до касира;
  • реалізацію плану проведення багаторівневих експериментів з метою отримання залежності середнього розміру черги та середнього часу очікування клієнта у черзі від кількості смуг та дисперсійний і регресивний аналіз отриманих результатів;
  • дослідження перехідного процесу для черги.
